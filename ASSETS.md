# External assets

Two large/copyrighted files are intentionally **not** committed to this repository
(the demo video exceeds GitHub's 100 MB file limit, and the paper is copyrighted).
They are hosted externally and referenced here.

## Demo video

`Group16-Major_project_demo.mp4` (~154 MB) — full walkthrough of PackageGuard.

- **Google Drive:** <https://drive.google.com/file/d/15ZeUebMfUJL_k0D4PuUrlwOokynzdlSV/view?usp=sharing>

> When sharing the Drive link, set access to "Anyone with the link can view".

## Source paper

*We Have a Package for You! A Comprehensive Analysis of Package Hallucinations by
Code-Generating LLMs* — Spracklen et al., USENIX Security 2025 (Distinguished Paper).

- **Official page:** https://www.usenix.org/conference/usenixsecurity25
- Search the USENIX Security '25 proceedings for the title above to access the PDF.

The local copy (`usenixsecurity25-spracklen.pdf`) is gitignored — download it from the
official source rather than redistributing it here.
